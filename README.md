# PyGREET v.s. GREET.Net Consistency Checking

This corresponds to the LCA results obtained by using PyGREET and GREET.Net for consistency checking and reconcilation between the two R&D GREET (Research and Development Greenhouse gases, Regulated Emissions, and Energy use in Technologies).

This is part of the supporting information for the manuscript *PyGREET: A modular Python platform of the GREET life
cycle analysis model for technology assessment* submitted to the journal [Environmental Modelling & Software](https://www.sciencedirect.com/journal/environmental-modelling-and-software).
